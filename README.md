#### Installation

Install Python dependencies:

```bash
pip install -r requirements.txt
```

Install NodeJS dependencies:

First:
```bash
cd client
```

Then:
```bash
npm install
```

#### Run

Run the server:

```bash
uvicorn main:app --reload --port 8000
``

Run the client:

First:
```bash
cd client
```

Then:
```bash
npm start
```
